Interfaces

CodeRed has a few built in interfaces that you can access via the console, with either imgui_open <interface-name> or imgui_toggle <interface-name>.

Changelog

Interface Name: changelog

The changelog window stores a hardcoded copy of every update that CodeRed has received, this interface also automatically opens after an update is released.

Dashboard

Interface Name: dashboard

Default Keybind: F1

The dashboard is the main interface used to control CodeRed, comparable to BakkesMod's settings window, it hosts most of the settings and commands available to the user.

ImGui Demo

Interface Name: demo

This interface is the default Dear ImGui demo window, used for debugging purposes, it showcases examples of every possible ImGui widget; especially useful when creating custom systles.

Post Process Editor

Interface Name: effects

The post process editor is a powerful tool used for creating real-time visual effects. It has a wide range of tools such as blurring, coloring, or lighting overrides, and works in offline modes and replay files. You'll have to play around with the settings on your own and see what you come up with, there are two examples by default that I created; one is a simple black and white filter and the other one is a green night vision thing.

GameState Debugger

Interface Name: gamestates

The gamestate debugger serves one purpose, debuggin which gamestate you are in. It displays your current playlist, active events, and permissions used by CodeRed.

Matchmaker

Interface Name: matchmaker

The matchmaker interface is a helpful little tool that allows you to quickly queue in different game modes at the end of matches.

Style Editor

Interface Name: styles

The style editor allows you to fully customize CodeRed's UI, you can change everything from widget rounding, spacing, custom colors, and even use your own custom fonts! If you would like to use your own fonts, they must be in .ttf format and placed in your CodeRed/Fonts folder prior to launching Rocket League.

Terminal

Interface Name: terminal

Default Keybind: F5

The terminal interface allows you to fully customize CodeRed via changing settings or executing commands, it also displays useful information for debugging purposes.

Thumbnail Renderer

Interface Name: thumbnails

The thumbnail renderer is an experimental interface used to render custom scenes for products, it also allows you to export them to a file of any format.

Playlist Tracker

Interface Name: tracker

The tracker interface is a little overlay that you can pin to one of the corners of your screen, it displays your current mmr, mmr needed to rank up/down, and win/loss streaks.

FAQs

What is CodeRed?

CodeRed is a highly customizable and easy to use third party enhancement for Rocket League. CodeRed aims to bring an "all in one", feature packed experience, to players with almost zero impact to performance.

Can I use CodeRed and BakkesMod at the same time?

Yes, but you might have to manually enable/disable MMR features in the mod you want to use; as if they are both enabled at the same time they WILL conflict with each other. The same goes for hotkeys and training features.

Can I customize my title with CodeRed?

You can equip titles that currently exist in the game, but you are not able to customize the text or color due to a request from Psyonix.

I found a bug or something doesn't work right.

CodeRed is currently in alpha so expect a lot of bugs and crashes, if you would like to report a bug or problem please do so in our Discord server.
