Changelog

Version 2.8.7 (December 9th 2025)

- Removed the interface_disable_roulette setting as its built into the game now.
- Added the new "Parc de Paris (Day)" map to Discord rich presence integration and the main menu background changer.
- Added the new "Mannfield (Quads)" map to Discord rich presence integration.
- Fixed the "Ranked Quads" playlist not displaying mmr needed to rank up or down in the tracker window.
- Experimental changes and bug fixes the custom avatar system, along with improvements to platform and steam avatars.
- Upgraded the internal ImGui framework to version 1.92.5.
- Updated the internal SDK to the latest version of Rocket League.

Version 2.8.6 (November 20th 2025)

- Added a new command interface_click_startmenu, which clicks past the start menu for you.
- Added a new setting autorun_delay, which is the delay in seconds before "AutoRun.cr" is called after injection.
- Added a new setting for the reflector window imgui_reflector_guid, which toggles between displaying FGuids as four integers or as strings.
- Fixed custom titles sometimes not showing up in the dashboard if you had more than eight created.
- Updated the internal SDK to the latest version of Rocket League.

Version 2.8.5 (November 8th 2025)

- Upgraded the internal ImGui framework to version 1.92.4, which changes how fonts are rasterized and rendered.
- Added over a dozen new setting and color options for custom styles that came with the new ImGui version.
- Improvements and crash fixes to users that had UTF32 characters in their computer or system path name.
- Safety improvements to parsing custom styles and applying colors.

Version 2.8.4 (November 1st 2025)

- Fixed an edge case crash that could happen between loading screens related to web requests.

Version 2.8.3 (October 29th 2025)

- Added a new setting imgui_scanner_flags, which will include function flags when copying to your clipboard or saving to a file.
- Added a new integer argument for the database_dump_titles command, database_title_sort_priority.
- Added a new "Database" tab in the custom title window, which has presets of the default title colors in game.
- The auto_purge_dumps setting now also removes crash dumps in "AppData\Local\CrashDumps\" in addition to "TAGame\Logs".
- Remade the ImGui image system internally, also added an auto cleanup system that removes leftover over icons.
- Fixed your title not restoring to its original one after deleting a custom one through the custom title window.
- Fixed the function table in the "Searcher" tab in the function scanner window not being scrollable.

Version 2.8.2 (October 24th 2025)

- Added a new setting imgui_terminal_cout which will also write all terminal text to "std::cout".
- Added a "version mismatch" detection system if you are in game and a Rocket League update is released.
- Fixed none of the custom chat commands working as of the last Rocket League update.- Fixed server side custom titles not working as of the last Rocket League update.
- Safety improvements to multi-threaded commands and terminal logging.
- Updated the internal SDK to the latest version of Rocket League.

Version 2.8.1 (October 17th 2025)

- When will psyonix stop breaking more than they fix every single update? This is their third hotfix now.
- Updated the internal SDK to the latest version of Rocket League.

Version 2.8.0 (October 14th 2025)

- The garage_apply_both setting will now be "true" by default for new users, this change does not effect current users.
- Internal optimizations to ImGui windows, as well as consistency changes to sizes and spacing of custom widgets.
- Updated the internal SDK to the latest version of Rocket League.

Version 2.7.9 (October 6th 2025)

- Fixed banners and borders not applying on the main menu as of the last update.
- Other minor miscellaneous bug fixes.

Version 2.7.8 (October 1st 2025)

- Updated the internal SDK to the latest version of Rocket League.

Version 2.7.7 (September 29th 2025)

- Added two new default themes, "Pink Bubblegum", and "Blueberry Lavender".
- Added customization support for the new "Ranked Quads" playlist in the "Textures" tab.
- Added a new setting imgui_terminal_history which lets you change the max user history for the terminal.
- Renamed the imgui_terminal_buffer setting to imgui_terminal_lines.
- Fixed Discord rich presence being stuck saying if you won or lost from the previous match if you entered an intermission game.
- Fixed a crash with the reflector window related to reading structs in TArrays that were in nested properties.
- Fixed a text wrapping mistake in the Terms Of Use window.
- Internal optimizations and memory improvements.

Version 2.7.6 (September 17th 2025)

- Added the new "Boostfield Mall (Day)" map to Discord rich presence integration and the main menu background changer.
- Fixed your CodeRed loadout not applying on the main menu if you were customizing your avatar border or banner in game.
- Fixed some instances where the goal explosion previewer could get stuck in the wrong camera state on the main menu.
- Updated the internal SDK to the latest version of Rocket League.

Version 2.7.5 (September 11th 2025)

- Fixed Discord rich presence integration not working for some users on Epic due to a thread racing issue.
- Multiple bug fixes and improvements to Discord rich presence related to ranks and the small text field.
- Multiple bug fixes related to the stadium_legacy_clubs setting for 1v1 matches.

Version 2.7.4 (August 31st 2025)

- Added a proper Terms Of Use and Privacy Policy to the mod, which will popup if you haven't agreed to it upon injection.
- Added a new "Main Menu" tab to the dashboard and moved the main menu related settings from the "Interface" tab to here, this is to make room for future features.
- Added an option for custom styles that change how color pickers are displayed, you can now pick between a hue bar or hue wheel.
- Fixed the CodeRed version number being wrong as of the last update.
- Other minor internal bug fixes and changes to the DirectX hook.

Version 2.7.3 (August 20th 2025)

- Added two new json keys for the population tracker, "OnlinePlayers" which are the total online players authed in, and "PlaylistPlayers" which are players actually playing in game modes (including offline and online).
- Added safety checks related to image sizes when loading custom textures, to prevent loading images that are bigger than what can actually be displayed.
- Fixed a crash that happened on injection if CodeRed tried to access file paths that had non UTF8 characters in them, such as emojis.
- Fixed titles not applying to profiles in custom workshop maps.
- Updated the internal SDK to the latest version of Rocket League.

Version 2.7.2 (July 29th 2025)

- Server side custom titles are now available as an exclusive perk to being a server booster in the Discord!
- Fixed a bug related to a rare case specific "special" titles where not showing up in the dashboard.
- Fixed float and array values not applying properly with the game_settings_import command.
- Experimental changes to when custom textures are destroyed between loading screens.
- Updated the internal SDK to the latest version of Rocket League.

Version 2.7.1 (July 11th 2025)

- Added five new settings that let you display the balls current speed under the "Offline" tab.
- Fixed the primary and accent color popups not working for loadout products in the dashboard.
- Fixed some tooltip descriptions for certain settings being wrong in the dashboard.
- Resized a lot of the text spacing and widget sizes in the dashboard to be more consistent.
- Internal stability improvements and safety measures related to custom textures.

Version 2.7.0 (June 27th 2025)

- Fixed the blacklist in the function scanner not applying if the whitelist filter was completely empty.
- Fixed not being able to paint your car body, which was related to last update to loadout codes.
- Miscellaneous fixes and improvements to various ImGui windows.

Version 2.6.9 (June 23rd 2025)

- Removed the garage_patch_invisible setting as its no longer needed as of the last Rocket League update.
- Fixed the interface_auto_limit_fps setting not restoring your original fps on next boot if you closed the game while it was unfocused.
- Fixed key binds to custom aliases or sequences not saving after restarting your game.
- Fixed a rare bug that caused custom banners to sometimes get recolored by the game.
- Minor internal bug fixes and safety improvements related to loadout codes.

Version 2.6.8 (June 17th 2025)

- For first time users, the dashboard and terminal windows will now automatically be opened upon first injection.
- Added the new "DFH Stadium (Anniversary)" map to Discord rich presence integration and the main menu background changer.
- Added a new setting imgui_scanner_limit, which lets you pick the limit of how many functions can be displayed in the function scanner at once.
- Changed the textbox in the sequence editor window to be a searchable combobox, so you don't have to manually type out commands or settings.
- Remade the custom alias and keybind windows both visually and internally, they now function similar to the new sequence window.
- Fixed not being able to call an already existing sequence from another custom sequence.
- Lots of code refactoring and changes to the internal framework.
- Updated the internal SDK to the latest version of Rocket League.

Version 2.6.7 (June 6th 2025)

- Added a new setting imgui_scanner_origin, which lets you see where a function was called from (ProcessEvent, ProcessInternal, or CallFunction).
- Remade the function scanner window both visually and internally to be much more optimized, as well as added a new "Settings" tab to it.
- Remade the sequence window in the "Bindings" tab to be easier to use, sequences let you bind multiple commands or actions to a single custom command.
- Fixed a crash that happened if you used any of the custom chat commands in modes that didn't have a chat, such as freeplay or custom training.
- Fixed a rare bug that caused CodeRed to think your account was cabined before you logged into the game.
- Additional internal memory optimizations and bug fixes.

Version 2.6.6 (May 23rd 2025)

- Added new context menu strips for properties in the reflector window, which let you copy their name, offset, and value.
- Added a new setting flip_indicator_unlimited_color which lets you pick the color of the flip indicator if you have unlimited flips enabled.
- Remade the matchmaker window internally to be fully dynamic and update playlists in real time, instead of relying on hardcoded playlist ids and names.
- Updated the internal API to accommodate GitHub's new rate limits of only one request per minute, this fix issues with appearing offline due to no API responses.
- Fixed the function scanner not splitting strings by commas properly as of the last update.
- Internal memory improvements, as well as optimizations to ImGui rendering code.

Version 2.6.5 (May 13th 2025)

- Added a new setting imgui_reflector_no_tarray, which disables reading from TArrays completely in the reflector window.
- Fixed a crash with the reflector window related to calculating missed offsets for some specific structs.
- Fixed the imgui_reflector_no_read setting not working in some scenarios.
- Updated the internal SDK to the latest version of Rocket League.

Version 2.6.4 (May 9th 2025)

- Renamed the texture browser window from textures to texture_browser.
- Renamed the manager_display_injection setting to imgui_notification_injection.
- Fixed a rare bug related to calculating split screen players mmr at the start of games.
- Fixed a crash with the thumbnail renderer related to the thumbnail_auto_garbage setting.
- Fixed a bug with the session tracker saying casual mmr instead of ranked for certain playlists.
- Fixed training pack codes not showing up in the "Automation" tab in the dashboard properly.
- Fixed CodeRed failing to validate global objects if you had hundreds of thousands of items in your inventory.
- Various other bug fixes and improvements related to the thumbnail renderer.

Version 2.6.3 (May 2nd 2025)

- Added the new "Force Field" map and "Car Wars" playlist to custom Discord Rich Presence info.
- Added two new commands, play_random_sound and play_random_sound_async, which pick a random ".wav" file from a given folder and plays it.
- Added a widget in the "Settings" tab for the imgui_reflector_array_limit setting, which sets the limit for reading and displaying values from TArrays.
- Added a new context menu option for TArrays in the reflector window that let you copy the contents of TArrays to your clipboard.
- Fixed the search box in the texture browser window being case sensitive when it shouldn't be.
- Fixed the interface_menu_background setting not resetting to default properly.

Version 2.6.2 (April 28th 2025)

- Fixed a bug causing custom player names to not send a filter request, causing them to never apply.
- Internal code refactoring and optimizations related to injection and initializing classes.

Version 2.6.1 (April 22nd 2025)

- Added custom avatars back as an experimental feature, if you encounter any crashes please report them to the Discord server.
- Fixed a crash that happened after closing the game, causing log info to be printed to your clipboard.
- Fixed the CodeRed logo in custom Discord Rich Presence sometimes using holiday logos for wrong dates.
- Fixed the reflector window not finding instances if you didn't toggle a setting off an on first.
- Fixed some quirkiness with custom player names on the main menu.

Version 2.6.0 (April 18th 2025)

- Added a new command database_dump_content, which converts and dumps the values from ContentMap to usable AES decryption keys.
- Added a "Settings" tab in the reflector window and converted all the checkbox options to actual settings, so they retain their value.
- Fixed some products rendering with either missing or wrong textures on them with the thumbnail renderer.
- Remade the "Discord Blurple" style to match Discords new default color scheme.
- Lots of internal code refactoring and bug fixes.

Version 2.5.9 (April 4th 2025)

- Added two new context menu options for the reflector window, you can now can copy an instances object index or its full name.
- Added a new setting `thumbnail_load_delay` which lets you change the delay to wait for loading product assets.
- Fixed textures exported as the ".jpg" format being partially corrupt due to the wrong color channels being written.
- Fixed a rare crash that could happen after loading to the main menu then quickly entering the garage.
- Fixed a crash with texture browser related to UTexture2D objects that had specific LOD Groups set.
- Fixed "FScriptDelegate" property values not being read for classes instances in the reflector window.
- Fixed ImGui windows not remembering if they were open from the last time you injected CodeRed.
- Fixed the `freeplay_unlimited_flips` setting not applying to your dodge time as well.
- Fixed custom color headlights not working on the main menu.
- Minor bug fixes and improvements to the thumbnail renderer.
- Backend changes to the internal API system.

- Added an "All" option for paints in the thumbnail renderer, which exports all paint variants of the selected product at once.
- Added two new commands play_sound and play_sound_async, which plays a ".wav" audio file from the given arguments.
- Removed the product_bool_include_se argument from the product dumper as it no longer exists.
- Various improvements and bug fixes to the new thumbnail renderer system.

- Fixed a bug that caused CodeRed to say you didn't meet the required specs related to grabbing your ram size.

Version 2.5.5 (March 7th 2025)

- Significant memory optimizations and improvements related to custom textures in addition to API requests.
- Fixed end of game automation features still being called if you joined another game before the delay was finished.
- Fixed CodeRed switching to offline mode if an update was released while you were already injected.
- Fixed some text cutting cut off in the bindings window in the dashboard for custom sequences.
- Fixed trails not working in game for CodeRed loadouts (it's about time).

- Remade how custom styles are parsed and written related to custom terminal themes and accents.
- Fixed the connections window sometimes not showing all available linked accounts when searching.
- Fixed the "[UTF16]" option not working properly for the database dumper commands.

- Renamed the database argument online_attribute_string to online_attribute_names for consistency reasons
- Fixed the database dumper not escaping some specific special characters, turns out Psyonix likes to add random escape characters to strings for no reason.
